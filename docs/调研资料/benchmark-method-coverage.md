## 备注

- Benchmark 变体已统一合并：如 `LoCoMo` 包含 `LoCoMo-10`、`LoCoMo` 等表述；`LongMemEval` 包含 `LongMemEval-S`、`LongMemEval-s`、`LongMemEvalS` 等；`SWE-Bench` 包含 `SWE-Bench-Verified`、`SWE-bench Lite`；`AIME` 包含 `AIME 2024`、`AIME 2025` 等。
- Composite benchmark（如 LongBench、RULER、InfiniteBench 等）保持整体不拆分。
- `MT-Bench` 和 `MT-Bench-101` 作为不同 benchmark 分别统计。
- `MSC` 统一指代 Multi-Session Chat 及其变体（含 Deep Memory Retrieval 子任务）。
- `ARC` 统一包含 ARC-Easy、ARC-Challenge 及其同类变体。
- `NaturalQuestions` 包含 `Natural Questions`、`NaturalQA`、`NQ` 等表述。
- `ETT` 统一包含 ETTm1、ETTm2、ETTh1、ETTh2 等变体。
- `MSRVTT`/`MSVD` 统一包含 QA 和非 QA 版本。
- `ActivityNet` 统一包含 ActivityNet-QA。
- `Ego4D` 统一包含 Ego4D-VQ3D、Ego4D NLQ、QA-Ego4D 等变体。
- `Minecraft` 统一包含 MineRL、Minecraft Universe Benchmark 等 Minecraft 生态评估。
- 原始文档中记作 `TopicoQA` 的条目，已修正为正确名称 `TopiOCQA`。
- 原始文档中记作 `ScreenshotVQA` 的条目，其核心 benchmark 为 `ScreenQA`。
- `GVD` 全称 "Generated Virtual Dialog"，来自 MemoryBank 论文（Zhong et al., 2024），并非独立的 benchmark 论文。
- `Smallville`、`Blocksworld`、`Gripper`、`Tyreworld`、`Barman` 等为环境/仿真名，非 benchmark 论文名。
- `AIME` 为已有数学竞赛，非专门数据集论文。
- `CC`（Common Crawl, <https://commoncrawl.org>）和 `Ubuntu IRC` 为公开语料库/数据集，非 benchmark 论文。
